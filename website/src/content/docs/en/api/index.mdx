This section documents the public Python API of Bub's core modules.

| Module | Description |
|--------|-------------|
| [Hook Specs](/en/api/hookspecs/) | All pluggy hook contracts — the extension surface of Bub |
| [Framework](/en/api/framework/) | `BubFramework` — the central runtime and plugin manager |
| [Skills](/en/api/skills/) | Skill discovery, validation, and registry |
| [Tools](/en/api/tools/) | `@tool` decorator and `REGISTRY` for agent tool registration |

:::note
API reference pages reflect the current source code in `src/bub/`. For hook semantics and plugin authoring, see the [Extension Guide](/en/extension-guide/).
:::
